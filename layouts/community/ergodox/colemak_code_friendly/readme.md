# ErgoDox EZ Code Friendly Colemak Layout

  This layout was inspired by qwerty_code_friendly layout. 

  Having all the symbols accessible without modifiers is a good improvement for coding.

## Keymap 0: Basic layer

```
.--------------------------------------------------.  .--------------------------------------------------.
|   _    |   !  |   @  |   #  |   $  |   %  |Bspc  |  |Enter |   ^  |   &  |   *  |   -  |   =  |   +    |
|--------+------+------+------+------+------+------|  |------+------+------+------+------+------+--------|
| Tab    |   Q  |   W  |   F  |   P  |   G  |   (  |  |  )   |   J  |   L  |   U  |   Y  |   ;  |   \    |
|--------+------+------+------+------+------|      |  |      |------+------+------+------+------+--------|
| Esc/L1 |   A  |   R  |   S  |   T  |   D  |------|  |------|   H  |   N  |   E  |   I  |   O  |  ' /L1 |
|--------+------+------+------+------+------|   [  |  |  ]   |------+------+------+------+------+--------|
| LShift |   Z  |   X  |   C  |   V  |   B  |      |  |      |   K  |   M  |   ,  |   .  |   /  | LShift |
'--------+------+------+------+------+-------------'  '-------------+------+------+------+------+--------'
| LCtl |Super | Alt  |   {  |   }  |                              | Left | Down | Up   |Right | Del  |
'----------------------------------'                              '----------------------------------'
                                     .-------------.  .-------------.
                                     | Ins  |CapsLk|  | Home | End  |
                              .------+------+------|  |------+------+------.
                              |      |      |  TG1 |  | PgUp |      |      |
                              | Tab  | Enter|------|  |------| Space| Bspc |
                              |      |      | Grave|  | PgDn |      |      |
                              |/LCtl |/Super|/LAlt |  |/RCtl |      |/RAlt |
                              '--------------------'  '--------------------'
```
