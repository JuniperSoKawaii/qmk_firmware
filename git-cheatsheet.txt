## QMK git

# one time setup original as remote "upstream"
git remote add upstream https://github.com/qmk/qmk_firmware.git

# update original "upstream"
git fetch upstream
# make sure it is on own forked master branch
git checkout master
# rebase master to merge remote upstream
git merge upstream/master
# finally push master changes back to own fork
git push -f origin master

# commit new changes
git commit -a -m ""
git push
