## 什么是 QMK 固件?

QMK (*Quantum Mechanical Keyboard*) 是一个开源软件社区。该社区支持像键盘，鼠标，MIDI设备的各种电子输入设备。社区的核心团体维护 [QMK 固件](https://github.com/qmk/qmk_firmware), [QMK 工具箱](https://github.com/qmk/qmk_toolbox), [qmk.fm网站](https://qmk.fm), 和这些文档。
