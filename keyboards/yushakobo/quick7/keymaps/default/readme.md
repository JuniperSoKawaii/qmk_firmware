# The default keymap for quick7

### Base Layer
```
Left Encoder(CW: Vol+, CCW: Vol-)
Right Encoder(CW: RGB HUE-, CCW: RGB HUE+)
,----------- --------- ------------,
|   MUTE    |  Func  |  RGB MODE  |
 ----------- --------- ------------
| Shift+Tab |   UP    |    TAB     |
 ----------- --------- ------------
|   LEFT    |  DOWN   |   RIGHT    |
`----------- --------- ------------'
```

### Func1 Layer
```
Left Encoder(CW: RGB HUE-, CCW: RGB HUE+)
Right Encoder(CW: Vol+, CCW: Vol-)
,----------- --------- ------------,
|   QK_BOOT   |  Func  | RGB TOGGLE |
 ----------- --------- ------------
|   HOME    |  VOL+   |    END     |
 ----------- --------- ------------
| MEDIA <<  |  VOL-   |  MEDIA >>  |
`----------- --------- ------------'
```
