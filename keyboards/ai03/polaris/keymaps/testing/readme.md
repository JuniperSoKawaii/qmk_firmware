# The PCB testing keymap for Polaris

LED control keys are placed on the numrow.  
  
ESC = Backlight Toggle  
1   = Backlight Step  
  
5   = RGB Toggle  
6   = RGB Mode Cycle  
7   = RGB Hue Increase  
8   = RGB Hue Decrease  
9   = RGB Saturation Increase  
10  = RGB Saturation Decrease  
11  = RGB Value Increase  
12  = RGB Value Decrease
