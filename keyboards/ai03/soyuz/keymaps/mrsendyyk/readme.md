## Build The Firmware
You will need to build the firmware. To do so go to your terminal window and run the compile command:

    qmk compile -kb ai03/soyuz -km mrsendyyk
See [The Complete Newbs Guide To QMK](https://docs.qmk.fm/#/newbs).
