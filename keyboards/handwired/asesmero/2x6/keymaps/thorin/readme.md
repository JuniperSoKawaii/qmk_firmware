# thorin 2x6 Layout

My personal layout

With this keymap you can use 33 shorcuts with a 2x6 keyboard.
The bottom left key is used to toogle layers.
you can add a neopixel in  pin F7 to see which layer is selected. (Feel Free to change the colors)
