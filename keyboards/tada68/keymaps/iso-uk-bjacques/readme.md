# TADA68 layout for ISO UK

Makes the following changes to the default iso-uk layout

## Base Layer

CapsLock -> Fn
Esc -> Esc `		(win + esc = `, shift + esc = Â¬)


## Function Layer

Delete -> Insert
PgUp -> Home
PgDn -> End

ijkl -> navigation keys
u -> home
o -> end
; -> delete

< -> vol down
> -> vol up
? -> mute

c -> backlight decreasen
v -> backlight increase
b -> backlight toggle
